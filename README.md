# TipToe 🛍️

TipToe is a modern e-commerce web application built with React, Vite, and TypeScript.  
It features product search, filtering, sorting, and a clean UI.

---

## 🚀 Features

- 🔍 Product search functionality
- 🛒 Product listing and details pages
- 🧩 Category filtering
- 💰 Price range filtering
- 🔃 Sorting (price, name, relevance)
- ⚡ Fast performance with Vite
- 🎨 Responsive UI with Tailwind CSS

---

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- React Router DOM
- Tailwind CSS
- React Icons

## ▶️ Getting Started

### 1. Clone the repository

```bash
git https://github.com/chriswebofficial/CELU.git
cd tiptoe

2. Install dependencies
npm install

3. Run the development server
npm run dev

Open your browser:

http://localhost:5173
4. Build for production
npm run build
5. Preview production build
npm run preview
🧠 Notes
Make sure your favicon is placed in:
public/favicon.ico
All product data is located in:
src/data/Product.ts
```
